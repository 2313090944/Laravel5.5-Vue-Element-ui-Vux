# Lravel5.5-Vue-Element-ui-Vux 环境搭建

## 安装过程

####1. 克隆项目
```
git clone git@github.com:MrWangwj/Laravel5.5-Vue-Element-ui-Vux.git
```

####2. 配置项目环境
1. 复制根目录中的.env.example文件为.env 文件
2. composer install  安装所需扩展
3. npm install  安装组件
4. npm run dev  编译组件
